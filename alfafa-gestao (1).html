🌿 Configurar Sincronização

Para que Rafael, Felipe e Reinaldo vejam os dados em tempo real, é necessário conectar ao Firebase (gratuito). Siga os passos abaixo uma única vez:

1. Acesse console.firebase.google.com e crie um projeto (ex: alfafa-gestao)
2. No projeto, vá em Firestore Database → Criar banco → Modo de teste
3. Vá em Configurações do projeto → Seus aplicativos → Web (</>) → Registrar app
